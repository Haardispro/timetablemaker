# Timetable Maker

**This is a cli timetable maker**



<u>To get started:</u>

Git clone this repository:

```shell
git clone https://www.github.com/Haardispro/timetablemaker
```

cd into this directory:

```shell
cd timetablemaker
```

Finally, run the program:

```shell
python3 timetablemaker.py
```
